Foi implementada uma Rede Neural RBF com 15 neurônios ocultos e largura gaussiana ($\sigma$) de 40. O treinamento utilizou a pseudo-inversa para cálculo dos pesos da camada de saída. A validação via hold-out (70/30) com 10 permutações demonstrou que o modelo é robusto, atingindo uma acurácia média de 87.53% com baixo desvio padrão, indicando boa capacidade de generalização para o problema de classificação da coluna vertebral.
